# フィラーカウンター

Google Docs のサイドバーで、ドキュメント内の日本語フィラーワード（「あの」「えーと」「なんか」など）をカウントするアドオンです。

## 機能

- ドキュメント本文に含まれるフィラーワードを自動検出・集計
- 発話者ごとの集計に対応（`発話者名: 発言内容` 形式の議事録に対応）
- 各フィラーの出現回数をバーグラフで表示

## 対応フィラー

あー / あの / あのー / あのう / いや / いやー / いやあ / うーん / うん / えーと / えと / えっと / えー / じゃあ / じゃー / すー / そのー / そのう / なんか / まあ / まー / んー

## 使い方

1. Google Docs でドキュメントを開く
2. メニューの「フィラーカウンター」→「フィラーをカウント」を選択
3. サイドバーに解析結果が表示される
