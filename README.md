# フィラーカウンター

**フィラーカウンター**は、Google Docs 上で動作するアドオンです。会議の文字起こしやインタビュー議事録に含まれる日本語のフィラーワード（「あの」「えーと」「なんか」など）を自動で検出・集計します。発話者ごとの傾向把握や、スピーチ改善のフィードバックにご活用ください。

## 機能

- ドキュメント本文に含まれるフィラーワードを自動検出・集計
- 発話者ごとの集計に対応（`発話者名: 発言内容` 形式の議事録に対応）
- 各フィラーの出現回数をバーグラフで視覚的に表示
- Google Docs のサイドバーからワンクリックで解析可能

## 対応フィラー

あー / あの / あのー / あのう / いや / いやー / いやあ / うーん / うん / えーと / えと / えっと / えー / じゃあ / じゃー / すー / そのー / そのう / なんか / まあ / まー / んー

## 使い方

1. Google Docs でドキュメントを開く
2. メニューの「フィラーカウンター」→「フィラーをカウント」を選択
3. サイドバーに解析結果が表示される

## アクセス権限について

このアドオンは、以下の権限を使用します。

| 権限 | 用途 |
|---|---|
| 現在開いているドキュメントの読み取り | フィラーワードの検出・集計のため、ドキュメント本文のテキストを読み取ります |
| Google Docs の UI 操作 | サイドバーおよびメニュー項目の表示に使用します |

**このアドオンは、ドキュメントの内容を外部サーバーに送信・保存・共有することは一切ありません。** すべての解析はお使いのブラウザ上で完結します。
